CST338: Software Design

Software Design, provides students with the fundamental concepts and  techniques for developing large-scale software systems. The course covers topics such as  object-oriented design, software design principles, software architecture, development  methodologies and software testing. The objective of this course is to give students the ability  to design and implement complex software systems, apply design principles to solve design  problems, model entire systems, apply agile development practices and test software to ensure  code quality. The lessons are key takeaways for any respectable computer science program that  will serve students well in technology-oriented careers.
